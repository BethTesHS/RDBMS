SQLly Command Tutorial

Below is a guide to the SQLly commands. Copy and paste these into the Console to try them out.

1. Managing Tables (Create & Drop)

You can create new tables to store specific data types.
Important: Every table must have a column named id of type int. Supported types are int and string.

-- Syntax: CREATE TABLE [name] (col1 type, col2 type) CREATE TABLE products (id int, name string, price int) -- Delete a table and all its data DROP TABLE products

2. Create (Insert)

Add new records to a table. Strings must be wrapped in quotes.

INSERT INTO products VALUES (1, "Gaming Mouse", 60) INSERT INTO products VALUES (2, "Mechanical Keyboard", 120)

3. Read (Select)

View data from your tables. You can filter by ID or join tables.

-- Select all columns SELECT * FROM products -- Filter by ID SELECT * FROM products WHERE id=1 -- Join (Example using users and orders) SELECT * FROM users JOIN orders ON users.id = orders.user_id

4. Update

Modify existing records. Note: You can currently only update by ID.

UPDATE products SET price=50 WHERE id=1 UPDATE products SET name="Wireless Keyboard", price=130 WHERE id=2

5. Delete

Remove specific records by their ID.

DELETE FROM products WHERE id=1
